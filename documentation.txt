PUID and PGID: Set the user and group IDs to match your system's user (usually 1000).

SERVERURL: The public domain or IP of your WireGuard server.

SERVERPORT: The port WireGuard will listen on.

PEERS: The number of client configurations to generate. You can adjust this based on the number of clients.

PEERDNS: DNS resolver to use for the clients.

ALLOWEDIPS: Defines the allowed IPs for the client to connect to through the VPN.

cap_add and sysctls: Ensure the container has the necessary capabilities to manage networking and VPN operations.
